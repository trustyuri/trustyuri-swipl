trustyuri-swipl
===============

This is a SWI Prolog library to generate and check _trusty URIs_.
See https://github.com/trustyuri/trustyuri.

This library is work in progress.


Dependencies
------------

The regex package has to be installed:

    ?- pack_install(regex).
